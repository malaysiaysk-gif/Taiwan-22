文法可能補語說不了話 (shuō bù liǎo huà)話す余裕がない状況的・体力的なキャパオーバー。「說不出話」は物理的に声が出ない状態。
文法可能補語不能 (bù néng)不可能、禁止ルールや物理的な制約による不可能。
文法結果補語走到一半 (zǒu dào yí bàn)半分まで歩いたところで「到（到達）」が補語。動作がどこまで達したかを表す。
文法結果補語洗乾淨 (xǐ gānjìng)きれいに洗う洗う（動詞）＋きれい（形容詞・補語）。洗った結果どうなったかを示す。
文法方向補語跳過去 (tiào guò qù)スキップする、飛び越える動詞＋複合方向補語。テスト等で問題を飛ばして次へ進む時の定番表現。
文法助動詞能夠 (nénggòu)十分に～できる「能」の強調。条件が整って（努力の末に）到達できるニュアンス。
文法条件只要 (zhǐyào)～さえすれば「只要能夠了解...（理解できさえすれば）」のように条件を提示する。
文法構造助詞的 / 地 / 得 (de)〜の / 〜に / 〜する程度が的＋名詞(走路的聲音)、地＋動詞(大聲地講話)、動詞＋得＋補語(講得很大聲)
文法反語怎麼可能 (zěnme kěnéng)まさか！あり得ない！単なる疑問ではなく、強い否定や驚きの感情。TOCFLの聴解で頻出。
文法疑問詞為什麼 (wèishénme)なぜ（客観的）純粋な理由や原因を尋ねるニュアンス。
文法疑問詞怎麼 (zěnme)どうして / どうやって想定外の事態に対する驚き（How come?）、または主観的な方法（How?）。
文法比較(否定)沒有 (méiyǒu)〜ほどではない「A 沒有 B (那麼) 形容詞」。「那麼」を入れると感情が強調される。
文法受け身・因果因為...結果...被淋濕了 (yīnwèi... jiéguǒ... bèi línshī le)〜だから、その結果濡れてしまった「被」の後ろの動作主が省略された自然な受け身の形。「結果」は悪い結末に多用。
文法語順・処置先...，然後把[A]放進[B] (xiān..., ránhòu bǎ [A] fàngjìn [B])まず〜して、それから[A]を[B]に入れる「先〜然後〜」の順序と、特定のモノを移動させる「把構文」の王道パターン。
文法接続詞・並列不僅...，而且...也... (bùjǐn..., érqiě... yě...)〜だけでなく、その上〜でもある「不但...而且...」の洗練版。「也」は必ず主語の後ろに置く。
文法強調構文連... 都/也... (lián... dōu/yě...)〜でさえ〜だ極端な例を挙げて状況を強調する。「都/也」は動詞（否定）の直前に置く。
文法語順・移動從 [A] 騎車 到 [B] (cóng [A] qí chē dào [B])[A]からバイク/自転車で[B]へ行く手段（騎車）は目的地（到B）の前に置くのが中国語の黄金ルール。
文法数量の推定大概... [数量] 左右 (dàgài... [数量] zuǒyòu)だいたい〜くらい「大概」は数字の前、「左右」は後ろから数量を挟み込む。並べ替え問題の頻出罠。
語彙頻度總是 (zǒngshì)いつも
語彙頻度好幾次 (hǎo jǐ cì)何度も回数のみ。感情は前後の文脈で決まる。
語彙意図打算 (dǎsuàn)～するつもりだ「算」(suàn) のみだと「計算する」「もういい（算了）」になる。
語彙条件・時間才能 (cáinéng)〜して初めて〜できる中国語では「やっとできる」という条件の達成を示す。
語彙才能天分 / 天才 (tiānfèn / tiāncái)才能（ギフト）日本語の「才能がある」と言いたい時はこちらを使う。
語彙指示詞這麼 (zhème)こんなに目の前の状況や近い事象を指す（主観的）。
語彙指示詞那麼 (nàme)あんなに / そんなに過去や遠くの状況を指す。
語彙義務得 (děi)〜しなければならない口語で非常に強力な「have to」。発音（děi）のひっかけに注意。
語彙評価・加減算 ＋ [形容詞] (suàn)〜な方だ、〜と言える例：「算便宜（安い方だ）」。世間一般の基準に照らした評価・マイルドな断定。
語彙時間副詞剛 (gāng)〜したばかり動作の直後を表す。「剛結婚（結婚したばかり）」「剛到（着いたばかり）」など。
語彙時間量詞分鐘 (fēnzhōng)〜分間（時間の長さ）時刻の「〜分(fēn)」との区別に注意。移動時間やテストの所要時間を表す時もこれ。
語彙量詞(丸・粒)顆 (kē)〜個 / 〜粒丸いもの、粒状のもの（果物、飴、星、心など）を数える。
語彙量詞(塊)塊 (kuài)〜個 / 〜切れ / 〜塊塊状のもの、切り分けたもの（ケーキ、肉、石鹸など）を数える。
語彙量詞(細長)條 (tiáo)〜本 / 〜匹細長いもの（道、魚、ズボン、ロープなど）を数える。
語彙量詞(問題)題 (tí)〜問、〜題テストの問題（這題、下一題）を数える必須の量詞。
語彙量詞(店舗)家 (jiā)〜軒、〜店舗店やレストラン（這家店）、会社などの組織を数える必須の量詞。
会話食事不用了，謝謝 (bùyòng le, xièxiè)もう結構です追加注文などを断る時の丁寧な定番表現。
会話会計麻煩結帳 (máfan jiézhàng)お会計お願いします一番丁寧。「幫我買單」(bāng wǒ mǎidān) は不自然なので注意。
会話会計幫我結帳 (bāng wǒ jiézhàng)会計してください発音注意：「結(jié)」は二声、「帳(zhàng)」は四声。
会話好き喜歡 (xǐhuān)好き（万能）日常的な好き。「愛」(ài) を使うと少し重い。
会話不満麻煩 (máfan)面倒、迷惑絡み合って（麻）心が乱れる（煩）イメージ。
会話諦め(万能)沒辦法 (méi bànfǎ)しょうがない万能な「It can't be helped」。状況的にどうしようもない時。
会話諦め(派生)沒轍 (méichè)お手上げ、降参口語表現。「沒辦法」よりも「もう降参」のニュアンス。
会話諦め(派生)無能為力 (wú néng wéi lì)無力である書き言葉的。能力が及ばない悲劇的な響き。
会話許容沒關係 (méi guān xì)気にしない、大丈夫謝られた時の「しょうがないよ（許すよ）」という意味。
会話打ち切り算了 (suànle)もういいや、やめる不満や諦めで議論や思考を打ち切る時の表現。
会話消極的選択只好 (zhǐhǎo)～するより他にない他に選択肢がない時の「仕方なく～する」。
交通基本票 (piào)チケット全般乗り物以外（門票、電影票など）にも幅広く使う。
交通購入訂票 (dìng piào)予約する事前の確保（ネットや電話）。
交通購入買票 (mǎi piào)買う窓口などでその場での購入。
交通手続き取票 (qǔ piào)発券するコンビニ等で予約済みの券を受け取る時は「取票」。
交通手続き出示 (chūshì)提示する「請出示您的車票（切符をご提示ください）」。

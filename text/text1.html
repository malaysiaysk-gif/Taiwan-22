台湾華語の発音まとめ

クリックして音声を聴きながら、発音の基本をマスターしましょう！

1. 声調（4つのトーン）

四声（第一声〜第四声）

音の高低変化で意味が変わります。台湾では第三声を低く抑えるのが特徴です。

第一声: 高く平らに (mā / 媽)

第二声: 急激に上げる (má / 麻)

第三声: 低く抑える (mǎ / 馬)

第四声: 高いところから急降下 (mà / 罵)

軽声

他の音節に続けて軽く発音します。（例: 他的 tā de）

2. 声調の変化（変調）

第三声の変調

  • 第三声の前: 第二声に変わる（例: 你好 nǐ hǎo → ní hǎo）
  • 一、二、四声・軽声の前: 低いまま発音（半三声）

"不" (bù) の変調

第四声の前に来ると、第二声 (bú) に変わります。

3. 基本の母音

単母音 (7つ)

a o e i u ü er

複合母音

複数の文字をなめらかにつなげます。（ai, ei, ao, ou, ia, ie など）

4. 鼻母音

前鼻母音 (-n で終わる)

舌先を上の歯の裏側にピタリとつけます。「ヌ」と言うつもりで。（an, en, in...）

後鼻母音 (-ng で終わる)

舌先を上あごにつけず、口を開けたまま「ン」と言います。（ang, eng, ing...）

5. 注意すべき母音

"an" の読み分け (3通り)

  • 通常: 「アン」
  • ian の時: 日本語の「エン」に近く
  • üan の時: 「アン」と「エン」の中間

"e" の読み分け (2通り)

通常は「ウ」と「オ」の中間音。ei / ie / uei などの組み合わせでは「エ」に近くなります。

6. 子音と「i」の読み分け

子音一覧と発音の特徴

無気音(息を出さない)と有気音(息を出す)、そり舌音(舌を少し奥にそり上げる)を区別します。

"i" の読み分け (3通り)

  • 通常: 「イ」 (yi, bi など)
  • zhi/chi/shi/ri: そり舌にした時の自然な口で「イ」っぽく
  • zi/ci/si: 「イ」の口の形で「ウ」

7. ピンインのつづり書き換えルール

"y/w" で始まる音節

母音の前に子音がない時、yやwを使って書き替えます。

  • y (yu以外), yi → i
  • w, wu → u
  • yu → ü

"j / q / x" + "ü"

j, q, xの後にüが来る場合、上の「¨」を省略して u と書きます。
※「ウ」ではなく「口をすぼめたイ (ü)」で発音します。

真ん中の母音の省略

子音がつくときは、真ん中の "o" や "e" が省略されて書かれます。読む時は軽く補って発音します。

  • 子音 + iu (iou)
  • 子音 + ui (uei)
  • 子音 + un (uen)
